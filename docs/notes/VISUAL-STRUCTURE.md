# Visual Structure Note

Date: 2026-03-14

## Principle

Visual structure is part of cognitive structure.

Formatting is not cosmetic. It affects how humans parse, compare, and think with information.

## Practical implications

When presenting structured material, prefer:

- clear section breaks
- stable indentation
- correctly closed code fences
- visually distinct examples
- scannable bullet lists
- consistent heading hierarchy
- enough spacing to separate ideas cleanly

## Why it matters

For humans, cues like spacing, contrast, chunking, and visual rhythm are part of meaning.
If formatting is broken, the reasoning becomes harder to follow even if the underlying content is correct.

## Ygg relevance

This applies directly to:

- CLI output
- suggestion cards
- explain/help text
- docs
- prompts
- memory files

Ygg should aim to be both:

- machine-readable
- human-legible

Those are not opposing goals. They should reinforce each other.
